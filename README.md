## Brief

Reactive and responsive E-commerce demo site.
API: Dummy JSON

## Built With

React, JS, Tailwind CSS, Vite

## Getting Started

### Installing

1; Clone the repo:

```bash
git clone https://github.com/MegumiKim/max-value.git
```

2; install dependencies

```bash
npm i
```

### Running

```bash
npm run build
```
